Two years ago I decided to explore brainfuck language and realised that I needed
some tools to learn it properly. I started by implementing a compiler, which
translate brainfuck code to linux executable, then a interpreter. I also build a
small tool that allow a "conversion" from python3 code to brainfuck, and a 
debugger.
I do not use these tools anymore but I guess they can be useful.
